Explore All Modules

A detailed roadmap of what you’ll learn across 9 modules — from fundamentals to advanced NLP applications.

1Beginner~1 week

NLP

A gentle introduction to Natural Language Processing and key terminology you’ll use throughout the course.

You’ll Learn
  • What NLP is, common tasks, and real‑world applications.
  • Core terms: token, corpus, stopwords, lemma, stemming, vocabulary.
  • Overview of classic vs. neural approaches in NLP.
Outcomes
  • Comfort with core NLP vocabulary and task categories.
  • A mental map of the 9‑module journey.
2Beginner2–3 weeks

Text Processing and Regular Expression

Clean, normalize, and structure raw text for downstream modeling using robust preprocessing pipelines.

You’ll Learn
  • Normalization: casing, punctuation, Unicode, numbers, contractions, emoji.
  • Tokenization basics; sentence vs. word tokenization.
  • Stopword handling, lemmatization, stemming (when/why).
  • Regex essentials: classes, groups, quantifiers, lookarounds.
  • Building reproducible preprocessing functions.
Outcomes
  • Production‑ready cleaning scripts and regex patterns.
  • Properly prepared text datasets for modeling.
3Beginner1–2 weeks

Edit Distance and Applications

Measure string similarity for tasks like spell checking, fuzzy matching, and approximate search.

You’ll Learn
  • Levenshtein, Damerau–Levenshtein, Jaro–Winkler; character n‑gram cosine.
  • Dynamic programming intuition; pruning & thresholds.
  • Spell‑checking, deduplication, entity resolution workflows.
Outcomes
  • Implement string‑matching utilities for noisy, real data.
4Intermediate3–4 weeks

N‑gram, POS, NER

Learn classic linguistic features and sequence labeling techniques before neural models.

You’ll Learn
  • N‑gram language models; smoothing and perplexity.
  • POS tagging (HMM/CRF intuition), chunking basics.
  • Named Entity Recognition datasets, features, and evaluation.
  • Error analysis and domain shift considerations.
Outcomes
  • Build & evaluate simple statistical taggers and NER pipelines.
5Intermediate2–3 weeks

Word Sense Disambiguation (WSD)

Resolve meaning ambiguity using context, lexical resources, and supervised methods.

You’ll Learn
  • Dictionary‑based (Lesk), knowledge‑based (WordNet) approaches.
  • Supervised/semisupervised WSD; dataset considerations; evaluation.
  • Applications to search, QA, and dialog systems.
Outcomes
  • Choose and implement a WSD strategy for your domain.
6Intermediate2–3 weeks

Sparse Vector (Embedding)

Represent text with classic sparse features and learn how to engineer them effectively.

You’ll Learn
  • Bag‑of‑Words, n‑grams, TF‑IDF; pros/cons vs dense models.
  • Feature selection, weighting, scaling, and normalization.
  • Similarity metrics and retrieval with sparse vectors.
Outcomes
  • Deliver strong baselines for classification/retrieval tasks.
7Advanced3–4 weeks

Dense Vector (Embedding)

Move from sparse features to neural embeddings and contextual representations.

You’ll Learn
  • Word2Vec, GloVe; training objectives and intuition.
  • Contextual encoders (ELMo/BERT family); token vs sentence level.
  • Vector operations, similarity search, and evaluation.
Outcomes
  • Use modern embeddings for downstream tasks & retrieval.
8Advanced2–3 weeks

Application Using Sparse Vector

Build production‑ready systems powered by sparse features.

You’ll Learn
  • Text classification pipelines and error analysis.
  • BM25 and lexical search; inverted indexes; query expansion.
  • Evaluation: precision/recall@k, ROC/PR, offline vs. online.
Outcomes
  • Ship robust sparse‑based applications (search, tagging).
9Advanced3–4 weeks

Application Using Dense Vector

Leverage dense retrieval and semantic representations for high‑accuracy systems.

You’ll Learn
  • Semantic search and RAG basics; vector databases.
  • Zero‑shot transfer and cross‑encoders vs. bi‑encoders.
  • Latency/throughput tradeoffs, caching, and evaluation.
Outcomes
  • Design & evaluate dense‑based applications end‑to‑end.
