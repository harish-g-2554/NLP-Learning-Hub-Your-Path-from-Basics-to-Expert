Setting up Python for NLP

A beginner-friendly guide to installing Python, NLP libraries, and getting your environment ready for text processing.

1. Prerequisites

  • Basic understanding of Python syntax
  • A computer running Windows, macOS, or Linux
  • Internet connection to download packages

2. Install Python

Download the latest version of Python (3.10 or newer) from python.org.

  1. Run the installer
  2. Check the box Add Python to PATH
  3. Verify installation with: 
    python --version

3. Install pip (if not included)

python -m ensurepip --upgrade

4. Create a Virtual Environment

python -m venv nlp_env
source nlp_env/bin/activate  # macOS/Linux
nlp_env\Scripts\activate     # Windows

5. Install Essential NLP Libraries

pip install nltk spacy gensim scikit-learn pandas matplotlib

Optional extras for deep learning NLP:

pip install torch transformers

6. Download NLP Resources

For NLTK:

import nltk
nltk.download('punkt')
nltk.download('stopwords')
nltk.download('wordnet')

For spaCy (English model):

python -m spacy download en_core_web_sm

7. Test Your Setup

import nltk
from nltk.tokenize import word_tokenize
print(word_tokenize("NLP is fun!"))

8. Recommended Tools

  • Jupyter Notebook — interactive coding (pip install notebook)
  • VS Code — for editing Python scripts
  • Anaconda — alternative Python distribution with pre-installed packages
